# SupplierGPT

AI-powered email writer for supplier communication.

## Problem
Procurement teams waste 20% of their time chasing suppliers for updates.

## Solution
Generate contextual supplier follow-up emails with 3 tone variations:
- Formal escalation
- Polite nudge
- Urgent request

## Tech Stack
- Frontend: React
- Backend: FastAPI
- AI: Claude API (Anthropic)
